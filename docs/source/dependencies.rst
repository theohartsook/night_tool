Dependencies
============

Night Tools relies heavily on the Scipy ecosystem, as well as command line tools like GDAL, CloudCompare, and LAStools.

Python Standard Library
-----------------------
- os
- re
- shutil
- subprocess
- multiprocessing

Python packages
---------------
- geopandas
- matplotlib
- numpy
- pandas
- PIL
- scipy
- skimage


Other software
--------------
- CloudCompare
- GDAL
- LAStools
- PDAL
